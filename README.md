Welcome to the Vortexa Python SDK, building upon the [Vortexa API Documentation](https://docs.vortexa.com)


## Getting Started

#### Installation

```bash
$ pip install vortexa-sdk
```

#### Authentication

Set your `VORTEXA_API_KEY` environment variable, that's all.
